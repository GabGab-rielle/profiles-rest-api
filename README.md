# Profiles REST API

Profiles REST API course code.

Connect to vagrnt server  
`vagrant ssh`

Logout of vagrant server  
`exit`

Create virtual environment
`python -m venv ~/env`

Activate virtural environment  
`source ~/env/bin/activate`

Deactivate virtual environment
`deactivate`
